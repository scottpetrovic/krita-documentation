Polyline Tool
=============

.. figure:: images/polyline_tool/32px-Krita_tool_polyline.png
   :alt: images/polyline_tool/32px-Krita_tool_polyline.png

Polylines are drawn like :doc:`polygons <Polygon_Tool>`,
with the difference that the double-click indicating the end of the
polyline does not connect the last vertex to the first one.
